University Syllabi Inclusivity Project

Home Projects Work
← Back to Projects

Analyzed a dataset of 169 university syllabi for key inclusivity metrics. Determined which demographics of instructors were likely to use inclusive language using classification trees and impurity measures (weighted entropy). Communicated findings through various data visualizations created in R and published the results in a peer-reviewed journal.
